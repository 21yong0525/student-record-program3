# student-record-program3 : 학생 성적 프로그램3

## 개 요

https://github.com/21yong0525/student-record-program2

학생 성적 프로그램2에서는 Java collection framework를 활용하였으며,

이번 프로그램에서는 mysql을 활용하여 데이터를 관리하도록 개발하였습니다.
